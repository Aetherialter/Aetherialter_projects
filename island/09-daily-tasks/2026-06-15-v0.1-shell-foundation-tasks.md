## 任务 3：调整 Tauri 窗口基础配置

### 目标

从普通窗口改成更接近浮层的小窗口。

### 修改文件

```text
D:\Aether_island\src-tauri\tauri.conf.json
```

### 建议先改

```json
{
  "title": "Aether_island",
  "width": 480,
  "height": 120,
  "resizable": false
}
```

### 暂时不要急着改

```json
"decorations": false
"transparent": true
"skipTaskbar": true
"alwaysOnTop": true
```

这些可以作为任务 4 再做，避免一次同时引入多个变量。

### 验收

- 窗口尺寸变小
- 应用仍能启动
- `pnpm tauri dev` 能运行

## 任务 4：尝试无边框透明窗口

### 目标

在任务 1-3 稳定后，再加入：

```json
"decorations": false,
"transparent": true,
"resizable": false,
"alwaysOnTop": true,
"skipTaskbar": true
```

### 注意

- `transparent: true` 只是允许透明，不会自动让页面透明
- CSS 里 `html/body/#app` 也必须设置透明背景
- `decorations: false` 会让系统标题栏消失，窗口默认不可拖
- 可通过 `data-tauri-drag-region` 指定拖拽区域

### 验收

- 窗口无标题栏
- 背景不是 800x600 大黑框
- 胶囊/HUD 区域可见
- 不要求鼠标穿透
