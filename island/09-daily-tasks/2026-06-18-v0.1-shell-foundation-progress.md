## 已完成内容

### 1. 模板 UI 替换

Tauri 默认 greet 页面已替换为 Aether_island 胶囊 UI：

- 顶部状态组件
- 状态点
- 标题
- 副标题

### 2. hover 动画优化

动画从直接修改 `width` / `height` 改为主要使用：

```text
transform
opacity
background-color
box-shadow
```

并在 TypeScript 中用 `requestAnimationFrame` 合并状态切换，减少鼠标状态快速变化时的抖动。

### 3. 小窗口配置

窗口从默认 `800x600` 调整为：

```json
{
  "title": "Aether_island",
  "width": 480,
  "height": 120,
  "resizable": false
}
```

这一步只做小窗口，不引入 `transparent` / `decorations` / `alwaysOnTop` / `skipTaskbar`，避免一次引入太多变量。

```powershell

```

原因：该命令需要人工肉眼检查窗口效果，包括 hover 手感、小窗口大小和视觉观感。

用户后续需要验证：

- 窗口是否为 `480x120`
- 胶囊是否显示在窗口顶部
- hover 是否顺滑
- 鼠标快速移入移出是否抖动
